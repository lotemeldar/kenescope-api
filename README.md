# Kenescope API

🧠 Public API for tracking Israeli Knesset legislative activity in real time.

## 🚀 Endpoints

- `GET /api/bills`  
  → Returns all bills  
  → Supports `?subject=education` filtering

- `GET /api/mks`  
  → Returns MK list

- `GET /api/votes`  
  → Returns recent vote records

npm install
npm start
